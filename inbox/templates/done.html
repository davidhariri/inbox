You're all set

Inbox is ready. Connect your MCP client to start managing todos.

Claude Desktop

Add this to your Claude Desktop config file:

{
  "mcpServers": {
    "inbox": {
      "type": "streamable-http",
      "url": "{{server_url}}/mcp"
    }
  }
}

Claude Code

Run this command:

claude mcp add inbox --transport http {{server_url}}/mcp

ChatGPT

In ChatGPT settings, add an MCP server with this URL:

{{server_url}}/mcp
